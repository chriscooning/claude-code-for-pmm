# Competitive Analysis Workflow

Systematically analyze competitors and update positioning strategies.

## When to Use

- Competitive research requests
- Win/loss analysis
- Positioning updates
- Any task with category: `competitive`

## Inputs Required

- **Competitor**: Which competitor to analyze?
- **Analysis type**: Full analysis, feature comparison, win/loss, positioning update
- **Urgency**: Is this blocking a deal or launch?

## Workflow Steps

### Step 1: Gather Existing Intel

Check `Knowledge/competitive-intel/` for:
- Existing competitor profiles
- Recent win/loss reports
- Previous positioning docs

### Step 2: Research Framework

For each competitor, gather:
- **Product**: Features, capabilities, limitations
- **Positioning**: How they position themselves
- **Pricing**: Pricing model and tiers
- **Market presence**: Market share, customer base
- **Recent moves**: Product updates, marketing campaigns, partnerships

### Step 3: Compare & Differentiate

Create comparison matrix:
- Feature comparison
- Strengths/weaknesses
- Our differentiation points
- Win themes

### Step 4: Update Positioning

Based on analysis:
- Update battle cards if needed
- Identify positioning gaps
- Create win strategies
- Document key talking points

### Step 5: Share Findings

Create summary for:
- Sales team (battle card updates)
- Product team (feature gaps)
- Leadership (market intelligence)

## Success Criteria

- [ ] Comprehensive competitor profile
- [ ] Clear differentiation identified
- [ ] Win strategies documented
- [ ] Sales materials updated
- [ ] Key stakeholders informed
