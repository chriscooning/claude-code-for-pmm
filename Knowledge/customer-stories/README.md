# Customer Stories

Case studies, testimonials, and use cases for sales and marketing.

## Structure
- `case-study-[customer-name].md` - Full case studies
- `testimonials.md` - Customer quotes
- `use-cases.md` - Common use cases

## Usage

When creating sales or marketing materials:
1. Reference relevant customer stories
2. Extract quotes and proof points
3. Document new use cases as you discover them
4. Keep stories updated with latest results/metrics
