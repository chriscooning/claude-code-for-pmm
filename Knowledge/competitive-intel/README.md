# Competitive Intelligence

Store competitor profiles, win/loss analysis, and market intelligence here.

## Structure
- `competitor-[name].md` - Individual competitor profiles
- `win-loss-analysis-[date].md` - Win/loss reports
- `market-intelligence.md` - General market insights

## Usage

When conducting competitive analysis:
1. Create or update competitor profile files
2. Document win/loss learnings after deals
3. Track market trends and competitor moves
4. Reference these files when creating battle cards or sales enablement materials
