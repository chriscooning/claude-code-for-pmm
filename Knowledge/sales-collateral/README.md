# Sales Collateral

Sales decks, battle cards, one-pagers, and other sales materials.

## Structure
- `battle-cards/` - Competitive battle cards
- `sales-decks/` - Sales presentation decks
- `one-pagers/` - Product one-pagers

## Usage

When creating sales enablement materials:
1. Store final versions here for easy access
2. Reference when creating new materials
3. Keep versions organized by date or product
4. Link to these files in task notes for context
