# Technical Content

Technical documentation, API guides, and developer-focused content.

## Structure
- `api-docs/` - API documentation
- `developer-guides/` - Developer tutorials and guides
- `technical-blog-posts/` - Technical blog post drafts
- `code-examples/` - Code samples and examples

## Usage

When creating developer-focused content:
1. Store technical docs here for reference
2. Keep code examples organized
3. Reference when creating devrel content
4. Maintain version control for API docs
