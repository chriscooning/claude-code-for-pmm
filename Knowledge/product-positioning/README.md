# Product Positioning

Messaging frameworks, value propositions, and positioning documents.

## Structure
- `messaging-framework.md` - Core messaging framework
- `value-props.md` - Value propositions by audience
- `positioning-[product].md` - Product-specific positioning

## Usage

When creating content or sales materials:
1. Reference positioning docs for consistency
2. Update as product evolves
3. Align all external communications with positioning
4. Use for competitive differentiation
